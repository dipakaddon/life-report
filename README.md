# 🕉️ वैदिक जन्म कुंडली — Vedic Astrology Web App

**पंच-ग्रंथ ज्योतिष | Rule-Based | Hindi Only | Swiss Ephemeris**

## Features
- ✅ Swiss Ephemeris (Lahiri Ayanamsa) से सटीक ग्रह गणना
- ✅ 26 जीवन विषयों पर हिंदी भविष्यवाणी
- ✅ विशेष योग पहचान (मालव्य, नीचभंग, संग्रह, आदि)
- ✅ Vimshottari Dasha Timeline
- ✅ Print / PDF Download
- ✅ Auto-geocoding (OpenStreetMap)

## Tech Stack
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Python (Swiss Ephemeris) via subprocess

## Setup

```bash
# 1. Install Python dependency
pip install pyswisseph

# 2. Install Node dependencies
npm install

# 3. Run development server
npm run dev

# 4. Open http://localhost:3000
```

## Production Build
```bash
npm run build
npm start
```
